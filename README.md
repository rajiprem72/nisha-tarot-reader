# 🔮 Tarot Shakti by Nisha – Website

An elegant, responsive, and modern tarot reading website built for **Tarot Shakti by Nisha**.  
This project is designed to provide a premium user experience across **mobile, tablet, and desktop devices**.

---

## 🌟 Features

- 🎨 Grand full-screen banner with elegant design
- 📱 Fully responsive (mobile-friendly)
- 🧘 Tarot reading information (history, significance, reliability, remedies)
- 🧩 Modular structure for future expansion
- 💼 Service sections (readings, courses, sessions)
- ⚡ Fast loading and lightweight design

---

## 🧭 Sections Included

- **Hero Banner**
  - Eye-catching tarot-themed background
  - Call-to-action button (Book a Reading)

- **About Tarot Reading**
  - Introduction
  - History
  - Significance

  - Remedies (Parigarams)

  - Photo & Video Gallery
